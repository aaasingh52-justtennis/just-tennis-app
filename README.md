# Just Tennis League - Season 4

A free community tennis league web app for Dubai players.

## Features

- **Contact List**: Browse all registered players with filters for level, zone, availability
- **Ladder**: Live rankings based on match results
- **Rules**: Season 4 rules and playoff format

## Tech Stack

- **Frontend**: Vanilla HTML/JS
- **Database**: Supabase (PostgreSQL)
- **Hosting**: Vercel/Netlify (free tier)

## Season 4

**Dates**: January 1 - April 1, 2026  
**Players**: 80 registered  
**Format**: Singles, points-based ladder
